# data-analytics-portfolio
A collection of data analytics projects demonstrating SQL, Excel and data cleaning skills. This project explores the Video Game Sales dataset by cleaning the data, defining analytical questions, and creating visuals that summarize key patterns in the industry.

## Dataset
The dataset includes global video game sales with regional breakdowns (North America, Europe, Japan, and Other), reported in millions of units. It contains game titles, platforms, genres, publishers, release years, and sales figures.

## Questions Answered
- Which genre is the most common?
- Which platform is the most common?
- Which game is the best-seller for each region?

## Methods
- Cleaned the dataset to remove inconsistencies and prepare it for analysis.
- Sorted and filtered data to answer each analytical question.
- Extracted regional best-sellers using targeted sorting.
- Created supporting tables and Excel visuals for each question.
- Documented each step in the project notes for reproducibility.

## Visuals
- Genre frequency bar chart
- Platform frequency bar chart
- Regional best-seller sales chart (in millions)

All visuals are stored in the visuals/ folder.

## Insights
- Action is the most common genre, indicating broad developer output and consistent consumer demand.
- The DS is the most common platform, reflecting its large game library and global popularity.
- Wii Sports is the top seller in NA and EU, likely due to console bundling and mass-market appeal.
- Pokemon Red/Blue leads Japan, aligning with the franchise's cultural impact and early handheld adoption.
- GTA: San Andreas leads in Other regions, showing strong global appeal outside major markets.

## Reproducibility
All steps, from cleaning to analysis to visualization, are documented in the project notes.
