# Project 2: Blue Bikes Data Analysis
This project analyzes two years of Boston Blue Bikes trip and station data using Python, pandas, and Jupyter Notebook. The dataset includes trip-level and station-level information covering 2011-2013.

## Objectives
- Explore system-wide usage patterns and trip volume.
- Summarize station distribution across municipalities.
- Analyze user participation by type and gender.
- Evaluate trip duration patterns and round-trip frequency.
- Track bike-level movement across stations.
- Identify bikes visiting all Somerville stations and rank bikes by distinct stations visited.

## Tools & Techniques
- Python (pandas)
- Jupyter Notebook
- Grouping, merging, and aggregation.
- Summary table generation and metric computation.

- 'blue_bikes_analysis.html' - Full analysis (recommended for viewing)
