# Project 1: Video Game Sales Analysis
A data analytics project exploring the Video Game Sales dataset using SQL, Excel, and data cleaning techniques. The  project focuses on identifying common genres, common platforms, and rrgional best-sellers through structured queries and supporting visuals.

## Dataset
The dataset includes global video game sales with regional breakdowns (North America, Europe and Japan), reported in millions of units. It contains game titles, platforms, genres, publishers, release years, and sales figures.

## Itinerary
- Day 1-2: Reviewed the dataset, set up the environment, and explored the data structure.
- Day 3: Documented initial observations and identified cleaning needs.
- Day 4: Cleaned the dataset in Excel and prepared it for analysis.
- Day 5: Answered the three main questions in Excel and created early visuals.
- Day 6: Refined insights and improved documentation.
- Day 7: Added an additional visual and updated notes.
- Day 8: Built the first version of the README and finalized the insights.
- Day 9: Planned the SQL approach and outlined the queries to write.
- Day 10: Used SQL to replicate the Excel results (most common genre, platform, and regional best-sellers), and documented the queries in notes/.
- Day 11: Wrote full SQL queries to generate the data needed for visuals (all genres, all platforms, and a combined regional best-seller query using UNION ALL), exported results to CSV, created the final SQL-based visuals, and documented the workflow in day11/.

## Questions Answered
- Which genre is the most common?
- Which platform is the most common?
- Which game is the best-seller for each region?

## Methods
- Cleaned the dataset to remove inconsistencies and prepare it for analysis.
- Used Excel to sort, filter, and answer the three analytical questions during early exploration.
- Created supporting tables and Excel visuals to understand initial patterns.
- Wrote SQL queries to replicate the Excel results and validate findings.
- Wrote full SQL queries to generate complete datasets for visuals (all genres, all platforms, combined regional best-sellers).
- Exported SQL results to CSV and created final SQL-based visuals.
- Documented each step in the project notes for reproducibility.

## Insights
- Action is the most common genre, indicating broad developer output and consistent consumer demand.
- The DS is the most common platform, reflecting its large game library and global popularity.
- Wii Sports is the top seller in NA and EU, likely due to console bundling and mass-market appeal.
- Pokemon Red/Blue leads Japan, aligning with the franchise's cultural impact and early handheld adoption.
